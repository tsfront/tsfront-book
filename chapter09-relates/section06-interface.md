# （前后端）接口规范

> 随着前后端分离开发的流行, 接口规范的制定愈发重要. 一套良好的接口规范可以提升协作开发的效率, 减少沟通障碍.
>
> 通常采用 REST 方式来提供接口, 使用 [JSON](http://json.org/) 格式来传输数据.
