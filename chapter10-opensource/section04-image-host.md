# 免费图床

> 图床？image bed? 我也好奇过中文为什么这么翻译。通过查阅 Google，准确的英文对应应该是“image hosting”，直译“图片托管”。不过，图床倒也形象得很，就是放图片的地方嘛。我们知道前端项目需要引用图片的场合很多，在图片较多、不想占用本项目存储情况下，可以将图片上传至提供图片托管服务的平台，然后获取图片url，直接引用即可。一般免费服务会有使用上的限制。



参考链接：
